GIT HELP

  • git init : versionner un dossier avec GIT
  • git status : verifier l'etat d'un dossier versionne avec GIT et de ses fichiers
  • git add : ajouter un fichier
  • git commit -m "commentaire" : commiter un fichier
  • git commit -am "commentaire" : indexer et commiter un fichier a la suite (raccourci)
  • git push origin master : envoyer fichier sur GitHub (apres commit)
  • git pull origin master : recuperer un fichier depuis GitHub
  • git branch : verifier les branches presentes
  • git branch nom_branche : creer une branche nommee nom_branche
  • git checkout nom_branche : se placer sur une branche
  • git merge nom_branche : fusionner une branche nommee nom_branche avec la branche actuelle (sur laquelle on se trouve)
  • git stash : mettre de cote une modification (les fichiers ne prendront donc pas en compte la derniere modification en cours)
  • git stash pop : prendre en compte la modification en cours (mise de cote jusqu'a present)

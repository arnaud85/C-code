GIT HELP

  • Login : arnaud85
  • git init : versionner un dossier avec GIT
  • git status : vérifier l'état d'un dossier versionné avec GIT et de ses fichiers
  • git add : ajouter un fichier
  • git commit -m "commentaire" : commiter un fichier
  • git commit -am "commentaire" : indexer et commiter un fichier à la suite (raccourci)
  • git push origin master : envoyer fichier sur GitHub (après commit)
  • git pull origin master : récérer un fihier depuis GitHub
  • git branch : vérifier les branches présentes
  • git branch nom_branche : créer une branche nommée nom_branche
  • git checkout nom_branche : se placer sur une branche
